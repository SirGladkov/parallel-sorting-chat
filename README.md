# Клиент-серверный чат  
Курсовая работа по дисциплине **«Параллельные и распределённые вычисления»**  
Университет «Дубна»

## Описание
Приложение реализует обмен текстовыми сообщениями между несколькими пользователями через TCP/IP.

### Сервер
- Принимает подключения на `127.0.0.1:8888`
- Рассылает все входящие сообщения всем подключённым клиентам

### Клиент
- WinForms-приложение
- Подключение, отправка и приём сообщений в реальном времени

## Требования
- Visual Studio 2017+
- .NET Framework 4.8

## Запуск
1. Скомпилируйте решение в Visual Studio
2. Запустите `ChatServer\bin\Debug\ChatServer.exe`
3. Запустите один или несколько экземпляров `ChatClient\bin\Debug\ChatClient.exe`
4. Нажмите «Подключиться к серверу» и обменивайтесь сообщениями

> ✅ Проект не требует сторонних зависимостей.
